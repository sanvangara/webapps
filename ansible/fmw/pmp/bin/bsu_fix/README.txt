Content:
========
This patch contains fix for Smart Update Tool (bsu).

Patch Installation Instructions:
================================
Copy content of this zip file with the exception of README file to your SmartUpdate bsu directory (MW_HOME/utils/bsu).

Run the script as follows:

Windows:
Usage: bsu_update.bat <install or rollback>
Example: bsu_update.bat install

Unix:
Usage: . ./bsu_update.sh <install or rollback>
Example: . ./bsu_update.sh install
